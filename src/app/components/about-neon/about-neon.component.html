Neon Dashboard

The Neon Dashboard is a highly configurable, open-source data analysis application that allows users to explore massive, complex datasets using a collection of data visualization widgets in a common web-based environment. The Neon Dashboard is powered by the Next Century Component Library, an open-source JavaScript library that offers aggregated search and filter behavior along with customizable data visualization components that operate directly on your own datastores and integrate into any JavaScript application.
